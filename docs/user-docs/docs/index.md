# EstateKit CRM

Документация для пользователей CRM-системы EstateKit на базе Odoo 19.

## Разделы

- [Маркетинг](marketing/index.md) — продвижение объектов, размещения, тир-листы, AI-скоринг, бюджетирование

## Сборка

```bash
cd docs/user-docs
uv sync
uv run mkdocs serve     # локальный сервер
uv run mkdocs build     # статическая сборка
```
