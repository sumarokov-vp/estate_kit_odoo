# Бюджетирование

Учёт маркетинговых расходов и планирование бюджета.

## Формирование бюджета

Рекомендуемый подход: **10% от GCI** (валовой комиссии агентства).

```
Бюджет(мес) = Фиксированная база + GCI прошлого месяца × 10%
```

Пересмотр — ежемесячно, стратегический — ежеквартально.

!!! info "Планирование вне Odoo"
    Планирование бюджета ведётся в Excel / Google Sheets. В Odoo учитываются только фактические расходы — через поле «Стоимость» на каждом размещении.

## Распределение по каналам

| Канал | Доля бюджета | Характеристика |
|-------|-------------|----------------|
| Крыша.kz | 30–40% | Высокая конверсия (5–7%), лиды закрываются за 3–6 мес |
| Instagram/Facebook (реклама) | 20–30% | Дешёвые лиды, конверсия 1–3% |
| Instagram/Facebook (контент) | 10–15% | Фото, видео, рилсы |
| Telegram | 5–10% | Низкая стоимость, удержание аудитории |
| Email | 5% | Высокий ROI, требует базы |
| Резерв | 10–20% | Новые каналы, сезонные кампании |

## Учёт расходов

### Прямые расходы

Учитываются в поле **Стоимость** на размещении:

- Размещение на Крыша.kz (конкретный объект)
- Таргетированная реклама конкретного объекта

Сумма расходов по объекту = сумма стоимости всех его размещений.

### Общие расходы

Учитываются на уровне агентства, вне системы размещений:

- Фотосъёмка / видео
- Ведение Instagram-аккаунта
- Telegram-канал, бренд-маркетинг

## Маркетинг в прибыльности сделки

Прямые маркетинговые расходы по объекту вычитаются из комиссии **до** распределения между участниками сделки:

```
Комиссия сделки
  − Прямые маркетинговые расходы по объекту
  = База для сплита между участниками
    ├─ Листинговый агент
    ├─ Агент покупателя
    ├─ ISA (если привёл лида)
    └─ Агентство (company dollar)
        └─ Из этого покрывается общий маркетинг + операционка
```

## KPI

| KPI | Формула | Частота |
|-----|---------|---------|
| Cost per Lead | Расходы на канал / лиды из канала | Еженедельно |
| Cost per Deal | Общие маркетинг. расходы / сделки | Ежемесячно |
| Marketing ROI | (Доход − расходы) / расходы | Ежемесячно |
