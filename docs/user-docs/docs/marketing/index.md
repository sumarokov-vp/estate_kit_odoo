# Маркетинг

Маркетинговая платформа управляет продвижением объектов недвижимости: формирование пула активных объектов, отслеживание размещений по каналам, автоматическая ротация, контроль актуальности.

## Как это работает

Из общей базы объектов формируется **маркетинговый пул** (~100 объектов), которые активно продвигаются. Состав пула определяется автоматически на основе двух факторов:

1. **AI-скоринг** — автоматическая оценка объекта (цена, качество, маркетинговый потенциал)
2. **Тир-листы** — экспертная оценка агентов и тим-лида

Объекты в пуле размещаются на рекламных каналах (Крыша.kz, Instagram, Telegram и др.). Система отслеживает состояние размещений и уведомляет менеджеров о необходимых действиях.

## Вкладка «Маркетинг» на карточке объекта

Вся маркетинговая информация по объекту доступна на вкладке **Маркетинг** в карточке объекта:

- Список размещений с цветовой индикацией статусов
- Результаты AI-скоринга (Price Score, Quality Score, Marketing Score)
- Кнопка «Рассчитать скоринг» для запуска оценки

Теги объекта (например, «Marketing Pool») отображаются в шапке карточки.
