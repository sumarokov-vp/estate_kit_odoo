# AI-скоринг

Автоматическая оценка объектов на базе AI (Anthropic Claude). Анализирует данные объекта и выдаёт оценки для отбора в маркетинговый пул.

## Параметры оценки

Каждый объект получает три оценки по шкале 1–10:

| Оценка | Что оценивается |
|--------|----------------|
| **Price Score** | Конкурентность цены относительно рынка (10 = значительно ниже рынка) |
| **Quality Score** | Качество объекта: ремонт, планировка, локация, год постройки |
| **Listing Score** | Качество карточки: количество фото, описание, заполненность характеристик |

Цветовые индикаторы: 🔴 (1–3), 🟡 (4–6), 🟢 (7–10).

Плюс текстовое обоснование по каждому критерию.

## Как запустить скоринг

### Для одного объекта

1. Откройте карточку объекта
2. Перейдите на вкладку **Маркетинг**
3. Нажмите **Рассчитать скоринг**

### Для всех объектов

При расчёте маркетингового пула (**Объекты → Рассчитать пул**) AI-скоринг автоматически рассчитывается для объектов, у которых нет свежего скоринга (старше 14 дней).

## Настройка

В **Настройки → Estate Kit → AI-скоринг**:

- **API-ключ Anthropic** — ключ для доступа к Claude API
- **Модель** — модель Claude для оценки (по умолчанию: claude-sonnet-4-5-20250514)

!!! warning "Требуется API-ключ"
    Без настроенного ключа Anthropic API скоринг работать не будет.

## Связь с Marketing Pool Score

Средний AI-скоринг — основной компонент [Marketing Pool Score](pool.md). Формула:

**MPS = средний AI-скоринг × вес скоринга + бонус тир-листов × вес тир-листов**

Средний AI-скоринг также проверяется отдельно: если он ниже минимального порога (по умолчанию 4.0), объект не попадает в пул независимо от тир-листов.

Результаты скоринга отображаются на вкладке **Маркетинг** карточки объекта в виде таблицы с историей оценок.
