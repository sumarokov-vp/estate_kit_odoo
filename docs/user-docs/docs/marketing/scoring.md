# AI-скоринг

Автоматическая оценка объектов на базе AI (Anthropic Claude). Анализирует данные объекта и выдаёт оценки для отбора в маркетинговый пул.

## Параметры оценки

| Параметр | Что оценивается |
|----------|----------------|
| **Локация** | Район, инфраструктура |
| **Цена** | Сравнение с аналогами. Результат: ниже рынка / рыночная / выше рынка |
| **Качество объекта** | Состояние ремонта, планировка, этаж |
| **Качество контента** | Фотографии, полнота описания |

## Результаты

Каждый объект получает три оценки по шкале 1–10:

- **Price Score** — конкурентность цены
- **Quality Score** — качество объекта и контента
- **Marketing Score** — итоговый маркетинговый потенциал

Плюс текстовое обоснование каждой оценки.

Результаты отображаются на вкладке **Маркетинг** карточки объекта в виде прогресс-баров.

## Как запустить скоринг

1. Откройте карточку объекта
2. Перейдите на вкладку **Маркетинг**
3. Нажмите кнопку **Рассчитать скоринг**

!!! warning "Требуется API-ключ"
    Для работы скоринга необходимо настроить ключ Anthropic API в **Настройки → Estate Kit → AI-скоринг**.

## Настройка

В **Настройки → Estate Kit → AI-скоринг**:

- **API-ключ Anthropic** — ключ для доступа к Claude API
- **Модель** — модель Claude для оценки (по умолчанию: claude-sonnet-4-5-20241022)

## Связь с пулом

Скоринг — один из двух факторов формирования маркетингового пула. Еженедельно система пересчитывает скоринг объектов в пуле. Если средний балл опускается ниже порога — объект выводится из пула.

Для объектов уже в пуле дополнительно учитываются маркетинговые метрики: просмотры, клики, количество лидов.
