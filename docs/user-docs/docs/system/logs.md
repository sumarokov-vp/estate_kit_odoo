# Логи

Логи модуля Estate Kit доступны в **Недвижимость → Объекты → Логи**.

## Что логируется

Логи записываются автоматически при выполнении фоновых операций:

- **marketing_pool** — расчёт Marketing Pool Score, ротация пула

В будущем будут добавлены другие категории (синхронизация, вебхуки и т.д.).

## Поля

| Поле | Описание |
|------|---------|
| Время | Точное время записи (с миллисекундами) |
| Категория | Модуль/операция, создавшая запись |
| Уровень | Info / Warning / Error |
| Описание | Краткое описание события |
| Подробности | Детали (параметры, список объектов) |
| Объект | Связанный объект недвижимости (если применимо) |

## Фильтры

- По уровню (Info / Warning / Error)
- По категории
- Поиск по описанию
- Группировка по категории или уровню
